Sibling Selectors

Maybe it's pretty obvious, but the Sibling Selector selects a sibling. Duh..

All the elements in a webpage are part of a family. Sibling Selectors allow you to be more specific with your element selection, and makes it so you don't need to use .class or #id selectors.

We utilize sibling selectors in two ways:

General Sibling Selector

To use this selector, we'll use the ~ character.

Example:
h2 ~ p {color: red;}

Adjacent Sibling Selector

To use this selector, we'll use the + character.

Example:
li + li {background-color: blue;}

Pseudo Classes

W3Schools definition: A pseudo-class is used to define a special state of an element.

These are commonly used when designing links. We often use a:hover, a:visited, and a:active.
